### Microservices Starter Project
This Project using several Depedencies, based on Spring Boot:

 1. **Springfox**   
 2.  **Eureka Server**   
 3.  **Feign**    
 4.  **Zuul** 
 5.   **OAuth2 Server**
 6.  **Oauth2 Resource Server**

List Microservices:

 1. **Auth Service**
 2. **Discovery Service**
 3. **API Gateway**
 4. **Logging Service**
 5.  **Starter Project**
